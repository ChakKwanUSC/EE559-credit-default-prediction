# EE559-credit-default-prediction
Project for machine learning I - supervised learning

# description
This project uses supervised methods to predict whether credit card users will default in the upcomingpay period.

# how to run
1. Upload "UCI_Credit_Card.csv" to your google drive
2. Open the notebook with Google Colab
3. Update the file path in cell 2 to match the location of the file in your drive
4. Run all cells to see results

# requirements
Note: all requirements should be available in Colab already.
python3, Numpy, Pandas, Matplotlib
